# Alpine Connect

Welcome to Alpine Connect, your central hub for mountain adventures and community connection. This application is designed to help you discover, join, and manage outdoor tours, reserve equipment, and connect with fellow adventurers.

## Features

*   **Tour Discovery**: Browse a list of upcoming tours, view detailed information, and join adventures.
*   **Interactive Calendar**: See all scheduled tours on a monthly calendar view.
*   **Material & Gear Reservation**: Reserve equipment like climbing gear, kayaks, and safety kits for your trips.
*   **Community News Board**: Share updates and connect with the community through a collaborative sticky notes board.
*   **User Profiles**: Manage your profile and view your tour participation.
*   **Admin Dashboard**: A dedicated section for administrators to manage tours, user reservations, documents, and seed sample data.

## Tech Stack

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app) and built with the following technologies:

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Backend & Database**: [Firebase](https://firebase.google.com/) (Firestore, Authentication, Storage)
*   **Generative AI**: [Genkit](https://firebase.google.com/docs/genkit)
*   **Icons**: [Lucide React](https://lucide.dev/guide/packages/lucide-react)

## Getting Started

To get the development server running:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
